# Hotel Reservation System

## Overview

The **Hotel Reservation System** is a Java-based application developed to simplify hotel room booking and reservation management. It enables users to search available rooms, book rooms, cancel reservations, simulate payments, and view booking details. The system follows Object-Oriented Programming (OOP) principles and uses File I/O to store booking information for future reference.

## Features

* Search available hotel rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* Store booking records using File I/O
* Console-based user interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File I/O
* ArrayList
* Scanner Class

## Requirements

* Java Development Kit (JDK 8 or above)
* Java IDE (Eclipse, IntelliJ IDEA, VS Code, or NetBeans)

## How to Run

1. Save the source file as `HotelReservationSystem.java`.
2. Open Command Prompt or Terminal.
3. Compile the program:

   ```bash
   javac HotelReservationSystem.java
   ```
4. Run the program:

   ```bash
   java HotelReservationSystem
   ```
5. Use the menu to search rooms, make reservations, cancel bookings, view booking details, or exit the application.
